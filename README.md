# TripAdvisor Recommendation System — Information Retrieval Project

A text-based recommendation system built on TripAdvisor reviews, using BM25, TF-IDF, Transformer embeddings, and a Hybrid model to recommend similar places based purely on user reviews (no metadata used in model design).

## Project Structure

```
notebooks/
  1_Data_Preparation.ipynb          — Preprocessing & TF-IDF keyword signature extraction
  2_BM25_Baseline_Evaluation.ipynb  — BM25 baseline + Level 1 & 2 evaluation framework
  3_TFIDF_Similarity_Model.ipynb    — Model A: TF-IDF + Cosine Similarity
  4_Transformer_Embedding_Model.ipynb — Model B: Dense Embeddings (MiniLM)
  5_Hybrid_Recommendation_Model.ipynb — Model C: Hybrid BM25 + MPNet (final model)
data/
  reviews83325.csv     — Raw reviews (not committed, see .gitignore)
  Tripadvisor.csv      — Place metadata (not committed)
utils/
  evaluation.py        — Shared evaluation functions (DRY)
docs/
  implementation       — Initial planning document
  realisation          — Technical journal / process log
Report.md              — Final project report
```

## Setup

```bash
pip install pandas numpy scikit-learn rank-bm25 sentence-transformers scipy nbformat
```

## Running

Run notebooks **in order** (1 → 5) from the `notebooks/` directory:
```bash
cd notebooks
jupyter notebook
```

> **Important**: `prepared_reviews.csv` is generated by Notebook 1 and saved into `notebooks/`. It is excluded from git (see `.gitignore`). Run Notebook 1 first before any other notebook.

## Data

Raw data files (`*.csv`) are not committed to this repository. Place them in the `data/` directory before running.

## Key Results

| Model | Level 1 Error | Level 2 Error |
|---|---|---|
| BM25 (Baseline)             | 0.61 | 3.87 |
| TF-IDF + Cosine Similarity  | 0.60 | 4.78 |
| Transformer (MiniLM)        | 0.69 | 4.46 |
| **Hybrid BM25 + MPNet**     | **~0.62** | **~3.83** |
